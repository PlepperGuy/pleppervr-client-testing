# PlepperVR Client Testing

## Overview
This repository contains the PlepperVR testing modpack in Modrinth pack (.mrpack) format.

## Installation
1. Download Prism Launcher from https://prismlauncher.org/download
2. Import modpack: Add Instance -> Import from File -> Select PlepperVR_Test.mrpack
3. Launch the instance in Prism Launcher

## Information
- Minecraft 1.20.1 Forge
- 86 mods with VR support
- 8GB+ RAM recommended
- Java 17+ required
